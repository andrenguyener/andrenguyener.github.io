## Getting Started

1. Install and use the correct version of Node using [NVM](https://github.com/nvm-sh/nvm)

    ```sh
    nvm install
    ```

2. Install dependencies

    ```sh
    yarn
    ```

3. Start the development server

    ```bash
     yarn dev
    ```

4. Open [http://localhost:3000](http://localhost:3000) with your browser.
